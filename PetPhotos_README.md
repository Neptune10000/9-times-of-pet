## 1. 功能简介

### 🎯 核心能力

**宠物多状态图像生成器** 是一款基于万相2.7（ Wan2.7 ）AI模型的图像生成技能。只需提供一张宠物照片（猫咪或狗狗），即可自动生成 **9张不同场景/风格/情绪状态** 的高质量图片，并合成一张可直接用于社交媒体发布的 **3x3九宫格拼图**。

### ✨ 特色功能

| 功能 | 说明 |
|-----|------|
| **特征保留** | 确保生成的9张图片都能识别是同一只宠物 |
| **场景多样性** | 室内/户外/幻想三大类，共9种不同场景 |
| **风格变化** | 写实摄影/电影感/梦幻散景/杂志风等 |
| **情绪表达** | 慵懒/好奇/酷帅/乖巧/神秘等多种表情状态 |
| **一键成图** | 自动合成九宫格并保存到桌面 |

### 🐾 适用场景

- **社交媒体发布** → 小红书、Instagram、微信朋友圈
- **宠物写真集** → 商业拍摄参考、个人收藏
- **表情包制作** → 不同状态的可爱表情
- **贺卡/壁纸** → 节日主题、个性化壁纸

## 2. 快速开始

### 📌 前置条件

1. **宠物照片**（JPG/PNG/WebP格式）
   - 建议清晰无遮挡
   - 正脸或侧脸均可
   - 单张即可，多角度更佳

2. **API Key 配置**（已配置可跳过）
   ```bash
   export DASHSCOPE_API_KEY="你的阿里云百炼API Key"
   ```

### 🚀 使用方式

**方式一：在 WorkBuddy 中直接使用**

1. 确保 Skill 已加载（上传到 `.workbuddy/skills/` 目录）
2. 提供一张宠物照片
3. 等待自动生成9宫格
4. 桌面文件夹查看结果

**方式二：复制 SKILL.md 上传到万相大赛**

1. 获取完整的 `SKILL.md` 文档
2. 上传到比赛指定入口
3. 提交参赛作品

## 3. 使用流程

### 7步完整流程

```
Step 1: 提供照片      →  用户上传宠物原图
    ↓
Step 2: 特征分析      →  AI识别宠物类型/毛色/特征
    ↓
Step 3: 九宫格规划    →  确定9张图的场景/风格/情绪
    ↓
Step 4: 提示词生成    →  为每张图生成专属描述
    ↓
Step 5: AI生成       →  调用万相2.7生成9张图片
    ↓
Step 6: 合成九宫格    →  Python PIL 拼接3x3网格
    ↓
Step 7: 保存桌面      →  自动保存到 ~/Desktop/PetPhotos/
```

    ✓ 9张图片已生成
    ✓ 九宫格已保存到桌面

## 6. 常见问题

### Q1: 生成的图片和原图不像怎么办？

**A:** 可以尝试以下方法：
- 提供更多角度的照片作为参考
- 在提示词中更详细描述宠物特征（毛色纹路、眼睛颜色等）
- 选择性地重生成某几张不满意的图片

### Q2: API Key 如何获取？

**A:**
1. 注册阿里云账号：https://www.aliyun.com/
2. 开通百炼服务：https://bailian.contu.aliyun.com/
3. 创建 API Key 并配置环境变量

### Q3: 图片链接过期了怎么办？

**A:** 图片链接有效期为24小时。建议：
- 生成后立即下载到本地
- 桌面文件夹已有保存的九宫格

### Q4: 可以自定义9宫格的场景吗？

**A:** 可以！提供照片时说明你想要的场景，例如：
- "生成9张图，但全部是户外场景"
- "全部做成圣诞节主题"
- "只生成猫咪睡觉的状态"

### Q5: 支持狗狗吗？

**A:** 完全支持！无论是猫咪、狗狗还是其他宠物，只要提供清晰照片即可。

## 7. 技术参数

### 🤖 AI 模型

| 参数 | 值 |
|-----|---|
| **模型名称** | wan2.7-image |
| **生图方式** | 图生图（保留主体特征） |
| **生成数量** | 9张/次 |
| **分辨率** | 1K (1024x1024) 或 2K (2048x2048) |

### 🔧 拼图参数

| 参数 | 值 |
|-----|---|
| **拼图工具** | Python PIL/Pillow |
| **网格布局** | 3行 x 3列 |
| **单格尺寸** | 1024 x 1024 px |
| **分隔间距** | 20 px |
| **背景颜色** | 白色 (#FFFFFF) |
| **输出格式** | PNG (quality=95) |

### 📦 依赖环境

- Python 3.x
- PIL (Pillow)
- curl (下载图片用)
- 阿里云百炼 API

### ⚙️ 调用示例

```bash
# 1. 上传参考图到OSS
python scripts/file_to_oss.py --file pet.jpg --model wan2.7-image

# 2. 生成9张图片
python scripts/image-generation-editing.py \
    --user_requirement "生成9张橘猫不同状态写真" \
    --input_images "oss://xxx/pet.jpg" \
    --n 9 --size 1K --enable_sequential

# 3. 合成九宫格
python3 << 'EOF'
from PIL import Image
# ... 拼图代码
EOF
```
